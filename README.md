# ⚡ FisioActive

Sistem Manajemen Klinik Fisioterapi Modern & Terintegrasi yang dibangun menggunakan **Google Apps Script** dan **Google Sheets** sebagai database.

## ✨ Fitur Utama
*   **Role-based Access Control**: Tersedia akses login untuk **Admin** (11 menu manajemen lengkap) dan **Terapis** (7 menu penanganan).
*   **Public Patient Check**: Pasien dapat mengecek status antrean jadwal mereka dan riwayat rekam medis langsung dari halaman awal menggunakan **ID Pasien** (e.g. `FA001`) atau **ID Rekam Medis** (e.g. `MR001`) tanpa harus login.
*   **Manajemen Komprehensif**: Dashboard analitis, pendaftaran pasien baru, jadwal harian, rekam medis digital, performa terapis, jenis terapi, dan laporan statistik.
*   **Desain Premium & Responsive**: Interface indigo flat yang responsif di HP dengan micro-animations.
